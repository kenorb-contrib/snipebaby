VirtueMart Importer Addon for JoomShopping README

Description:
Easy migrate to JoomShopping from VirtueMart.
Absolutely full import of all supported by JoomShopping data, options and configuration from VirtueMart.
Flexible setup of import process.
English and russian translations.
=========================================
Version:
v1.5 [2011-09-06]
=========================================
Compatibility:
- PHP 5.2/5.3
- Joomla 1.5
- JoomShopping 2.9.6
- VirtueMart 1.1.9

Install:
Not using standart Joomla method. In admin area Components / JoomShopping / Install & Update
=========================================
Using:
In admin area Components / JoomShopping / Options / Import & Export / VirtueMart Import
=========================================
Import WORKS for:
- Advanced attributes (JShop: Attributes)
- Categories
- Type parameters (JShop: Characteristics)
- Coupons
- Custom attributes (JShop: Free Attributes)
- Manufacturers
- Products
  - Items attributes
  - Files
  - Relations
  - Reviews
  - Shipment prices
- Shipping methods
- Store info
- Taxes
- User groups
- Users extended data
- Vendors
